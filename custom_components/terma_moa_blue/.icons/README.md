# Terma MOA Blue Logo

To add the official Terma logo to this integration:

1. Download the Terma logo from: https://en.termaheat.com/images/logo_terma_red_horizontal_2.svg
2. Convert to PNG (256x256 or 512x512)
3. Save as `icon.png` or `icon@2x.png` in this directory
4. Restart Home Assistant

The logo will appear in:
- Integration list
- Device cards
- Configuration pages
